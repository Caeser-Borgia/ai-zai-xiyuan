第一章 · 从吉他出发的基础乐理

这一章从你会的吉他和弦出发，把这首歌需要的乐理全部讲通。

1.1 音名与唱名

音乐只有12个音，循环重复：

C C# D D# E F F# G G# A A# B （然后回到C） │ │ │ │ │ │ │ │ │ │ │ 哆 升哆 来 升来 咪 发 升发 嗦 升嗦 啦 升啦 西

每个相邻音之间 = 半音（吉他上一个品）。

两个半音 = 全音（吉他上两个品）。

🎸 记住：吉他一品 = 半音。全音=两品。这个最重要。

1.2 大调音阶

大调音阶的公式：全-全-半-全-全-全-半（W-W-H-W-W-W-H）

比如C大调：

C → D → E → F → G → A → B → C 全 全 半 全 全 全 半

这首歌是B大调，B大调的音阶是：

B → C# → D# → E → F# → G# → A# → B 全 全 半 全 全 全 半

注意看：B大调有5个升号（C#、D#、F#、G#、A#），不熟没关系，你只需要知道：

  • 这首歌的\"家\" = B和弦
  • 歌里大部分和弦都从这7个音里长出来

1.3 和弦是怎么长出来的

一个和弦 = 从音阶里隔一个取一个音，叠起来。

大三和弦（比如B）：

B 大调音阶：B — C# — D# — E — F# — G# — A# — B 隔一个取一个：B — D# — F# 这就是 B 和弦：B-D#-F#

小三和弦（比如G#m）：

G# — B — D# 也是隔一个取一个，但G#到B是3个半音（小三度）不是4个半音（大三度）
🎸 吉他上按一个B和弦（x24442），松开食指让Bass变成A→就是B7。加一个小指在1弦4品→就是Bmaj7。和弦都是这样\"加一个音\"变出来的。

1.4 B大调的和弦家族

从B大调音阶的每个音上各建一个和弦，得到：

级数和弦构成音类型颜色
IBB-D#-F#大三🎯 主和弦（家）
IIC#mC#-E-G#小三次要角色
IIID#mD#-F#-A#小三温柔色彩
IVEE-G#-B大三\"离开家\"的感觉
VF#F#-A#-C#大三⬆️ 属和弦（想回家）
VIG#mG#-B-D#小三忧伤色彩
VIIA#mA#-C#-E#小三很少用
🎯 这首歌主歌主要用 I → VI → IV → V（B → G#m → E → F#），记住这四个就够了。

第二章 · 这首歌的和弦全拆解

这首歌实际用到的和弦并不复杂——来，一个一个看：

2.1 大和弦（蓝色）B · E · F# · B♭

听感：明亮、稳定、快乐。大和弦是流行音乐的\"主力\"。

  • B（I级）：这首歌的\"家\"。每次回到B都感觉稳定。
  • E（IV级）：副歌和弦。听感像\"出去转了一圈\"。
  • F#（V级）：属和弦。听感\"不稳定\"→想要回到B。
  • B→E→F#→B 就是最经典的\"回家\"路径。

2.2 小和弦（绿色）G#m · D#m · Am

听感：忧伤、温柔、暗淡。小和弦让歌里有\"感情\"。

  • G#m（VI级）：主歌的主要小和弦，配合\"法典\"\"玄武岩\"的厚重感。
  • D#m（III级）：\"你在橱窗前\"——温柔的感觉。
  • G#m→E→D#m的走向：越来越暗，配合歌词从\"历史\"到\"你\"的聚焦。

2.3 七和弦

七和弦 = 在普通和弦上再加一个音（七度音）。

  • Bmaj7 （紫色） = B + A#。比B多了一层\"梦幻感\"。
  • #Gm7 （绿色） = G#m + F#。七度让声音更\"流动\"，不那么死板。
  • E♭7 （橙色） = 属七和弦。自带\"紧张\"属性，用在过渡段制造悬念。

2.4 转位和弦（斜杠和弦）

比如 B/#F = 按B和弦，但最低音弹F#。

为什么要这样做？让低音线条更顺。

B/#F → #F/#D → #Dm7 → 低音一路向下：F# → D# → D# 如果都用原位：B → F# → D#m → 低音跳来跳去，不顺耳
🎸 转位和弦在吉他上不太好按，但听感非常细腻。林迈可大量使用这个技巧。

2.5 这首歌的所有和弦速查表

Bmaj7大七B-D#-F#-A#梦幻感
#Gm7小七G#-B-D#-F#忧伤流动

第三章 · 和弦进行（Chord Progressions）

和弦进行 = 和弦的排列顺序。就像走路时左脚右脚交替——不同的顺序产生不同的\"步伐感\"。

3.1 这首歌的\"1645\"

主歌的和弦循环：B → G#m → E → F#

用级数表示就是：I → VI → IV → V（1-6-4-5）

B (I) → G#m (VI) → E (IV) → F# (V) 主音（家） 小和弦（忧伤） 大和弦（离开） 属和弦（想回家）

这是一个非常经典的R&B走向。为什么好听？因为它形成了一个\"情绪弧线\"：

  • B = 稳定（家）
  • G#m = 略带忧伤
  • E = 明亮（放松）
  • F# = 紧张（想回家）→ 回到B

3.2 副歌的和弦变化

副歌用了一个不同的进行：III → VI → II → V（3-6-2-5）

#Dm7 (III) → #Gm7 (VI) → #C (II) → #F (V) → B (I)

这个进行比主歌更\"不稳定\"。III级和II级都不是\"家\"，一直在\"往外走\"，最后才回到B。配合副歌\"我给你的爱写在西元前\"放开旋律，形成了\"旋律往上冲、和声在外飘\"的效果。

3.3 为什么1645这么好用？

I-VI-IV-V（1645）是全世界流行音乐最常用的进行之一：

  • 它包含了三个大和弦（I、IV、V）和一个小和弦（VI）
  • 大和弦给你稳定感，小和弦给你情感，V级属和弦制造\"回家\"的推力
  • 这首歌把1645拉长到2-4小节一个和弦，营造\"叙事感\"而非\"推进感\"
🎯 用你会的吉他试试：按 B → G#m → E → F#，感受一下这个\"情绪弧线\"。

第四章 · 转调（Modulation）

转调是这首歌最牛的地方。来，彻底搞懂它。

4.1 转调是什么？

一首歌大部分时间待在一个调里（这首歌是B大调）。转调 = 整首歌突然搬家到另一个调。

想象你在B大调的房间（家具都是B、G#m、E、F#），突然搬到了B♭大调的房间（家具变成了B♭、Gm、E♭、F）。

家具变了，但\"摆放的方式\"（I-VI-IV-V的结构）是一样的。

4.2 这首歌的三次转调

B大调（主歌/副歌）→ B♭大调（主歌重复）→ C大调（桥段）→ B大调（副歌回来）

第一次转调：B → B♭（降半音）

  • 位置：第二遍主歌，\"古巴比伦王颁布了…\"
  • 手法：硬切（Direct Modulation）——没有过渡和弦直接从B跳到B♭
  • 效果：像\"陷入更深的回忆\"
  • 为什么特别？99%的流行歌升调（为了更激动），这里降调（为了更深沉）

第二次转调：B♭ → C（升大二度）

  • 位置：桥段\"我感到很疲倦\"
  • 此处转调后进入一个全新的调（C大调），和弦变成F→C→F→G→Am
  • 效果：像\"另一首歌插了进来\"，创造时空撕裂感
  • 这段是录音室临时加上的，原Demo没有

第三次转调：C → B

  • 桥段结束后用了一个G7过渡和弦平滑转回B大调
  • G7 = C大调的V级 = B大调的bVII级，一拍的时间就完成了过渡
  • 你完全察觉不到，但情绪上\"回到了熟悉的地方\"
🎯 听歌时的感觉：桥段\"疲倦\"那几句感觉\"世界安静了\"\"情绪往下坠\"——这就是降调+减法编曲的力量。

第五章 · 编曲入门

很多人分不清作曲、编曲、制作——这里是区别：

5.1 三个角色

角色做什么这首歌是谁
作曲写出旋律+和弦框架周杰伦
编曲决定用什么乐器、怎么搭配、前奏多长、哪里加弦乐林迈可
制作人把控整体方向、监督录音和混音周杰伦
  • 作曲 ≈ 写剧本
  • 编曲 ≈ 选演员+导演怎么演
  • 制作人 ≈ 制片人

5.2 林迈可做了什么

林迈可（Michael Lin）是周杰伦的\"编曲三巨头\"之一。这首歌他的核心贡献：

  1. 前奏Lo-Fi设计：用低通滤波+磁带饱和做\"古老录音\"效果
  2. R&B节奏编配：用轻快的鼓组消解历史题材的沉重感
  3. 分层进入：前奏鼓先→吉他后，不是所有乐器同时出现
  4. 乐器选择：竖琴、锡塔尔、击弦古钢琴——为\"古文明\"服务
  5. 声场规划：前奏窄→主歌中→副歌宽，全程有设计

注意：周杰伦的Demo已经把旋律、吉他Riff、和声框架写好了。林迈可把血肉填上去做成成品。

5.3 编曲的核心思维

好的编曲不是\"越多越好\"，而是为歌词服务：

  • 主歌密集歌词 → 编曲做减法，让你听清字
  • 副歌\"爱写在西元前\" → 弦乐+和声全部进来，情感爆发
  • 桥段\"疲倦\" → 所有乐器抽走，只剩钢琴+人声
  • 结尾\"一切又重演\" → 直接淡出，没有告别
🎯 编曲的最高境界：你感觉不到编曲的存在，只感受到情绪。

第六章 · 乐器图鉴

这首歌用了14种不同的乐器/音色。以下是快速认识它们：

🎸 吉他（核心乐器）

  • 角色：贯穿全曲的Riff，这首歌的灵魂
  • 音色：Clean Tone + 压缩 + 短混响
  • 怎么听：前奏那个\"当当当\"就是它，整首歌几乎一直在弹

🥁 鼓组

  • 角色：节奏基底，R&B风格
  • 构成：Kick（底鼓）+ Snare（军鼓）+ Hi-Hat（镲）+ Clap（拍手）
  • 怎么听：听\"咚·啪·咚·啪\"的节奏，这就是R&B的backbeat

🎹 击弦古钢琴（Harpsichord）

  • 角色：色彩点缀，句尾闪一下
  • 音色：不是钢琴——它像\"拨弦\"的声音，巴洛克时期乐器
  • 为什么用：巴洛克=古老 → 绑定古文明意象

🎻 弦乐组

  • 角色：副歌加入，增加厚度和史诗感
  • 怎么听：副歌\"我给你的爱\"——背景那种\"嗡——\"的长音就是弦乐

🎵 竖琴（Harp）

  • 角色：间奏Solo+背景装饰
  • 怎么听：间奏那段像\"流水\"一样的声音
  • 为什么用：最古老的乐器之一 → 绑定古文明

🎵 锡塔尔（Sitar）

  • 角色：异域色彩
  • 音色：印度乐器，带鼻音的嗡嗡声
  • 注意：混音中几乎听不见，更多是心理暗示

🎹 合成器

  • 角色：氛围铺底，填满声场
  • 怎么听：前奏那种\"空灵感\"就是它在工作

🎤 人声

  • 主歌：rap-like单音吟唱，像在\"念\"历史书
  • 预副歌：\"诗篇\"叠了三轨和声，声场突然变宽
  • 副歌：旋律完全打开，音域跳五度
  • 桥段：混响增加，像在一个空旷的大厅里唱歌
🎯 练习：第一遍只听吉他，第二遍只听鼓，第三遍只听人声——这样你的耳朵会越来越敏感。

第七章 · 制作技术入门

以下是一些制作相关的术语，用最简单的方式解释：

7.1 Lo-Fi

Lo-Fi = Low Fidelity（低保真）。故意让音质变差。

  • 前奏那段\"闷闷的\"吉他就是Lo-Fi处理
  • 用了低通滤波（把高频切掉）和磁带饱和（模拟老旧磁带的声音）
  • 不是设备不好——是故意这么做的，为了营造\"古老\"感

7.2 Filter Sweep（滤波扫描）

前奏14秒内，cutoff频率从2kHz慢慢推到全频——声音从\"闷\"变\"亮\"。

寓意：尘土褪去，石碑上的字显现。

7.3 混响（Reverb）

混响 = 声音在房间里反射产生的回声效果。

  • 主歌：混响小 → 像在室内说话
  • 副歌：混响大 → 像在大厅里唱歌
  • 桥段：混响更大+空旷 → 制造\"疲倦\"\"远离家乡\"的孤独感

7.4 声场（Stereo Field）

声场 = 声音从左到右的宽度。

  • 前奏：很窄（像从收音机里放出来）
  • 主歌：中等宽度
  • 副歌：很宽（左中右三个方向都有声音）

全程的声场变化：窄 → 中 → 宽 → 窄（桥段）→ 宽

7.5 压缩（Compression）

压缩 = 把声音最大和最小的差距缩小。

作用：让人声更\"稳\"，你听到的每个字音量差不多。

7.6 Pedal Tone（持续音）

低音一直停在同一个音上，上方和弦在变化。

副歌的F#低音持续了4个小节。效果：低音\"踩在大地上\"，上方\"仰望星空\"。

🎯 用播放器的0.75x或0.5x速度听，更容易听到这些细节。

第八章 · 怎么听？练习指南

按照以下顺序，每遍只听一个维度：

🎯 第一遍：只听歌词

关注方文山怎么把\"古巴比伦\"\"汉摩拉比\"\"楔形文字\"这些词放进节奏里。

注意\"祭司·神殿·征战·弓箭\"——每个词一个重音，像在石碑上刻字。

🎯 第二遍：只听吉他

前奏的Riff是整首歌的灵魂。注意它几乎贯穿全曲不变。

什么时候吉他变清晰了？什么时候它停了（桥段）？

🎯 第三遍：只听鼓

前奏第5小节鼓才进入。注意R&B的\"弹跳感\"节奏。

桥段鼓变稀疏——注意\"疲倦\"的感觉和鼓的关系。

🎯 第四遍：只听贝斯

贝斯的低音线条。注意副歌部分低音停在F#上不变。

🎯 第五遍：只注意和弦变化

打开播放器的和弦模式，看着和弦听。

什么时候是B（稳定）？什么时候是F#（紧张）？什么时候转调了？

🎯 第六遍：只用学习播放器的L1模式

看乐器标签 + \"为什么好听\" 注释。

🎯 第七遍：L2模式

加入和弦级数和编曲手法。

🎯 第八遍：L3模式

看完整的和声分析和制作细节。

🎯 人的大脑一次只能专注一个维度。一遍只听一个东西——坚持下来，一个月你的耳朵会比现在敏感十倍。
