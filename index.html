🎧 用 UVR5 提取分轨

下载 Ultimate Vocal Remover 5（免费开源）：

GitHub: github.com/Anjok07/ultimatevocalremovergui

提取步骤：

  1. 打开 UVR5 → 选择 MP3 文件
  2. Model 选 MDX-Net → Kim Vocal 2（最佳平衡）
  3. Output 选 Vocals + Instrumental
  4. 提取一次得到人声和伴奏
  5. 伴奏再跑一次：选 MDX-Net → UVR-MDX-Net Main
  6. 得到：鼓/贝斯/其他

提取后把文件放入 HTML 同目录的 stems/ 文件夹：

stems/vocals.wav
stems/guitar.wav
stems/drums.wav
stems/bass.wav
stems/strings.wav
stems/keys.wav
